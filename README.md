To Run Local--

----------
FrontEnd :
  clone the my-app UI code and open in any VDI like visual code studio
  make sure your system contain NPM and node to run locall
  after cloning and opening code in VDI Please run below commands 
  npm i -f   // to install the node modules
  npm run dev   // it will start the local on server port http://localhost:5173/

----------

Backend :
 clone code Booking in your filefolder and open in any VDI your comfortable Eclipse/STS/intellij
 make sure your added JAVA class path in your system
 open main class file BookingApplication.java and right click or hover on main to run the class
 it will start the backend server local host with port 8080 
 // http://localhost:8080/
